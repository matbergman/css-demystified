# CSS Demystified
How to Love CSS3. My presentation, with source files, about best practice mobile-first CSS. 

## Topics
* Semantic HTML5
* Naming with BEM
* Preprocessor strategies with LESS and SASS
* CSS Specificity
* Mobile-first CSS.
